# qrcore-generator

### *What is?*

* This code, make a QR Code based in the informations added in the ``requirements.txt`` file

### *usage*

* ``git clone https://github.com/emanuelvsz/qrcode-generator.git``
* ``cd qrcode-generator``
* edit the file ```requirements.txt`` how do you like
* ``go run ./main.go``
